# OGC NewFinity Platform — Documentation FAQ

Frequently asked questions for developers and contributors working on the OGC NewFinity Platform.

---

## 1. Where do I find all system diagrams?

All diagrams are located in:

docs/mermaid/

The index for all diagrams:

docs/mermaid/index.md

---

## 2. What naming rule must all diagrams follow?

Use this format:

phase-X[.Y]--category--topic.mmd

Examples:

- phase-7.6--wallet--architecture.mmd  
- phase-9.5--amy--tool-map.mmd  

---

## 3. Where do I add new diagrams?

Add new diagrams into the correct phase folder within:

docs/mermaid/

Then update:

- The folder's README
- docs/mermaid/index.md

At:

docs/developer-onboarding.md

docs/index.md

## 7. Where are Cursor Agent rules stored?

In:

.cursor/context/

## 9. What if I find outdated or incorrect documentation?

- Maintain formatting
- Keep links valid
- Follow naming rules
- Add entry to docs/changelog.md
