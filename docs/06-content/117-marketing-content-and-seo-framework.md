# OGC NewFinity — Marketing Content & SEO Framework for the Platform (v1.0)

## 1. Introduction

This document defines the marketing content strategy, SEO standards, messaging framework, distribution channels, and brand communication guidelines for OGC NewFinity.

The framework ensures:

- Clear brand storytelling
- High-quality content output
- Strong online discoverability
- Optimized user acquisition funnels
- Consistent messaging across all channels

This applies to the website, platform pages, landing pages, onboarding materials, blogs, challenge promotions, and all SEO-driven content.

## 2. Marketing Objectives

### 2.1 Core Goals

- Increase platform visibility
- Improve user acquisition
- Boost retention through high-value content
- Strengthen brand identity
- Improve search rankings
- Support global expansion

### 2.2 Key Pillars

- Clarity
- Consistency
- Creativity
- Transparency
- Value creation

## 5. SEO Framework

### 5.1 Technical SEO Requirements

- Proper H1/H2/H3 hierarchy
- SEO-friendly URLs
- XML sitemaps
- Robots.txt rules
- Fast-loading pages
- Mobile-friendly design
- Schema markup (future integration)

### 5.2 On-Page SEO Requirements

Each page must include:

- One clear H1
- Descriptive metadata
- Targeted keywords
- Short paragraph introductions
- Internal links
- Image alt text
- Optimized headings

### 5.3 Content-Length Guidelines

- Landing pages: 600–1500 words
- Blog posts: 1200–2500 words
- Tutorials: 500–1500 words

## 6. Keyword Strategy

### 6.1 Keyword Categories

- Challenges (STEM, AI, Art, Creativity)
- Education & Learning
- Innovation & Technology
- Global Youth Programs
- Digital Skills Development

### 6.2 Keyword Selection Process

- Research target keywords
- Analyze search intent
- Identify competition level
- Select primary & secondary keywords
- Integrate naturally into content

### 6.3 Keyword Placement

- Page title
- H1 heading
- First paragraph
- Subheadings
- Image alt text
- URL slug
