This document defines how to prepare the OGC NewFinity documentation for conversion into a static website using tools like Docusaurus, MkDocs, VitePress, or Next.js Docs Mode.

---

## Requirements

- A clean folder structure

- Section-based organization

- Consistent naming conventions

- Markdown-first documentation

- Dedicated sidebar structure

---

## Migration Steps

1. Choose documentation framework.

2. Generate configuration files.

3. Move /docs content into website structure.

4. Generate sidebar automatically or manually.

5. Deploy to GitHub Pages, Vercel, or Netlify.
