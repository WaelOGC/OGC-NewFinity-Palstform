# Sidebar Configuration Guidelines

This document explains how to build the sidebar configuration for documentation frameworks like Docusaurus and MkDocs.

---

## Rules

- Use folder-based grouping.

- Order items logically.

- Include all main documentation files.

- Auto-generate sidebar when possible.

---

## Recommended Sidebar Sections

- Overview

- Quickstart

- Structure Map

- Mermaid Index

- API Section

- Test Documentation

- RFCs

- Glossary

- CI/CD

- Maintenance
