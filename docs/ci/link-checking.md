# Link Checking Rules

Future CI/CD will run automatic link validation.

---

## 1. Internal Links

Ensure:

- Docs correctly reference each other

- Mermaid index links work

- Folder READMEs point to valid diagrams

---

## 2. External Links

Rules:

- Must use HTTPS

- Must not link to expired or inaccessible domains

- Avoid direct file hosting links unless approved
