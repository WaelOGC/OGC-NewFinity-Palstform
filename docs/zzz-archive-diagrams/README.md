# Archived Diagrams

## Archive Policy

Diagrams are archived (not deleted) when they:
- Contain old token allocation models
- Use outdated roadmap phases
- Reference pre-OGCFinity naming
- Use old Challenge Program structures
- Include outdated architecture models
