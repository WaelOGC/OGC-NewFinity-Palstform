# OGC NewFinity — Admin Dashboard Modules (Vision Document)

**Version:** v1.0

**Status:** Vision / Planning Document — Not Implemented

**Maintained by:** OGC Technologies

**Last Updated:** 2024

> **Note:** This document outlines planned admin dashboard modules. All modules described herein are **Planned / Not Implemented** unless otherwise noted.

---

## 1. Purpose

This document describes the planned modular structure of the Admin Control Panel dashboard. Each module represents a distinct functional area for administrative operations.

All modules are conceptualized as part of the future Admin Control Panel vision and have not yet been implemented.

---

## 2. Module Overview

The Admin Control Panel is envisioned as a collection of specialized modules, each focused on specific administrative tasks. Modules will be shown or hidden based on user role and permissions.

**Current Status:** Vision Only — No modules implemented beyond basic admin user management.

---

## 3. Planned Admin Modules

### 3.1 User Management Module

**Status:** Planned / Not Implemented

**Scope:** Internal Admin Use (Future)

**Planned Capabilities:**

- **User Search & Filtering:**
  - Search by email, username, user ID
  - Filter by role, status, subscription tier
  - Filter by registration date, last activity
  - Advanced search with multiple criteria

- **User Profile Viewing:**
  - Complete user profile information
  - Activity history and timeline
  - Contribution history
  - Badge collection
  - Wallet balance and transaction history
  - Submission history
  - Challenge participation

- **User Actions:**
  - Role assignment and updates
  - Account status changes (active, suspended, banned)
  - Subscription management
  - Password reset initiation
  - MFA reset (future)
  - Forced logout capability

- **Bulk Operations (Future):**
  - Bulk role updates
  - Bulk status changes
  - Bulk email sending
  - Bulk export to CSV

**Access Roles:** Admin, Core Team, Founder
